<!-- USAGE EXAMPLES -->
## ‚å®Ô∏è Usage

* To deploy production app:
  * Build the app
  ```sh
    ng build
  ```
  * Deploy the app
  ```sh
    <simple server deployment> /dist/angular-video-player/.
  ```
  
 * To use development mode:
  * Deploy the app
  ```sh
    ng serve
  ```
  * Navigate to [localhost:4200](http://localhost:4200/)
